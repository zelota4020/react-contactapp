# --= contact app (frontend)=--

This is a simple contact application that allows you to add, edit, delete and view contacts.

### How it's created

```npm create vite@latest```                (react - typescript)
<br/>```npm i axios react-router-dom```
<br/>```npm install```

### How it starts

```npm run dev```
